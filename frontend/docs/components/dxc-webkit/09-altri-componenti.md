# Altri componenti dxc-webkit

Elenco degli altri componenti esportati dalla libreria. Per API dettagliate (props e tipi) consultare i file `.d.ts` in `node_modules/dxc-webkit/dist/components/<NomeComponente>/`.

| Componente | Descrizione sintetica |
|------------|------------------------|
| **Accordion** | Accordion con AccordionItem, AccordionHeader, AccordionBody. |
| **Alert** | Messaggio di avviso/info. |
| **Avatar** | Avatar utente (AvatarGroup per gruppi). |
| **BottomBar** | Barra inferiore. |
| **Breadcrumb** | Breadcrumb con BreadcrumbItem, BreadcrumbIcon. |
| **ButtonGroup** | Gruppo di pulsanti. |
| **Card** | Card con sub-componenti (CardHeader, CardBody, …). |
| **Carousel** | Carousel. |
| **Checkbox** | Checkbox. |
| **Chip** | Chip/tag. |
| **DataTile** | Tile per dati. |
| **DatePicker** | Selezione data. |
| **DefaultPage** | Layout pagina predefinita. |
| **Dropdown** | Menu a tendina. |
| **EmptyIcon** | Icona stato vuoto. |
| **FloatingPanel** | Pannello flottante. |
| **Footer** | Footer con partials. |
| **Form** / **FormGroup** | Wrapper form e gruppi. |
| **Header** | Intestazione pagina. |
| **HeroBanner** | Banner hero. |
| **Language** | Selezione lingua. |
| **Layout** | Box, griglia e layout (Box con padding, margin, flex, font, color, …). |
| **List** | Lista. |
| **Loader** | Loader generico. |
| **MegaboxFilter** | Filtro megabox. |
| **Navscroll** | Navigazione con scroll. |
| **Pagination** | Paginazione. |
| **Popover** | Popover. |
| **RadioButton** | Radio button. |
| **Scrollbar** | Scrollbar custom. |
| **Sidebar** | Sidebar (SidebarItem con Icon, label, isActive). |
| **Slider** | Slider (rc-slider). |
| **Stepper** | Stepper per wizard. |
| **Switcher** | Switch/toggle. |
| **TabNavigation** | Tab con Tab (TabNavigation, Tab). |
| **TextArea** | Area di testo multiriga. |
| **TimePicker** | Selezione ora. |
| **Toggle** | Toggle. |
| **Tooltip** | Tooltip. |
| **VideoViewer** | Visualizzatore video. |

## Export aggiuntivi

- **constants:** tipi e valori condivisi (colori, font, sizes).
- **functions:** `convertClickToKeyDown`, `debounce`.

Per l’uso nei componenti del progetto, fare riferimento ai documenti in questa cartella e ai tipi in `node_modules/dxc-webkit/dist/components/`.
