# Utility e cache

Documentazione delle librerie di utilità e cache usate nel backend (o disponibili nell’ecosistema MASE).

- **[mase-utils-cache — Cache Hazelcast](./mase-utils-cache.md)** — Decoratori `@cacheable`, `@cache_evict`, `@cache_put` per cache distribuita.
- **[mase-utils-commons — Utilità condivise](./mase-utils-commons.md)** — Serializzazione JSON ↔ oggetti, naming e tipi (`camel_to_snake`, `json_to_object`, `object_to_json`).
